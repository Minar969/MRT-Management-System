# MRT Management System

A metro rail application that helps operate and manage a metro rail system for a reliable user experience.

## Overview

This system provides comprehensive management capabilities for metro rail operations, including:
- System operation monitoring
- User experience optimization
- Rail network management
- Real-time operational control

## Technology Stack

- **Language**: C#
- **Platform**: .NET Framework

## Prerequisites

- .NET Framework/Core (specify version)
- Visual Studio 2022 or later (recommended)
- SQL Server (or your database)

## Getting Started

### Clone the Repository
```bash
git clone https://github.com/Minar969/MRT-Management-System.git
cd MRT-Management-System
```

### Build the Project
```bash
# Open the solution file in Visual Studio
# Or build from command line:
dotnet build "MRT Management SystemSln.sln"
```

### Run the Application
```bash
dotnet run
```
